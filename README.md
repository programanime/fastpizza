# Project
this is a demo project to order some pizza

# Requirements
- Java 17
- Node 20
- Apache Maven 3.9.0

# Deploy
To deploy the application you need to run this command: 
```
mvn spring-boot:run
```
Then you need to open this url [http://localhost:8080](http://localhost:8080)
